# Engineering Tool Matrix

> Every core concept of Negation-based Economics maps to a line of executable code in the engine. Every engine output maps to a citable formula in the textbook.

## Six Tools

| Tool | Domain | Core Function |
|:---|:---|:---|
| **LightSky Engine** | Asset Pricing | Infrastructure engineering calculator. 4-layer existence checks → FVR-TVC dual-axis pricing → Monte Carlo ROI simulation. |
| **RouteGuard** | Risk Pricing | Route risk assessment engine. 4D risk factors → 5000 simulations → risk scoring. Proactive actuarial methodology. |
| **YH Risk Ecosystem** | Insurance Actuarial | Risk exposure wholesale/retail + insuretech infrastructure. Receives RouteGuard output + AAP data → premium pricing → insurance/reinsurance/ILS. |
| **FaultLine Translator** | Translator | Multi-fault-zone universal translation engine. Terminology mapping + logic rules + data transformation. |
| **AAP** | Operations Data | Aircraft aftermarket platform. Continuous standardized maintenance data supply → actuarial input. |
| **Distillation Engine** | Data Foundation | Data aggregation / cleaning / standardization / logging. Immutable trust root for all tools. |

## Tool Synergy

- AAP → YH: maintenance data → actuarial input
- RouteGuard → YH: risk assessment → premium pricing
- LightSky → FLT: simulation output → translated output
- YH → ILS: actuarial results → bond pricing

**Pricing Infrastructure = LightSky + RouteGuard. Asset pricing and risk pricing are two faces of the same negation-based methodology.**
